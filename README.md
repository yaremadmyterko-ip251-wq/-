# Банківський демо-сайт для GitHub Pages

Простий статичний сайт українською мовою для демонстрації банківського рахунку:

- Реєстрація користувача.
- Збереження даних реєстрації (ім'я + логін) в `localStorage`.
- Поточний баланс.
- Поповнення та зняття коштів.
- Історія операцій.

## Як запустити локально

Просто відкрий `index.html` у браузері.

## Як викласти на GitHub Pages

1. Створи репозиторій на GitHub.
2. Завантаж файли цього проєкту в корінь репозиторію.
3. На GitHub відкрий **Settings → Pages**.
4. Обери гілку (наприклад, `main`) і папку `/root`.
5. Збережи налаштування — через 1-2 хв сайт буде доступний за URL GitHub Pages.
